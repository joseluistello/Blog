Hola!

Soy un analista de datos de México que ama la ingeniería de software, la escritura, el aprendizaje y los negocios.

Mis dos principales lenguajes de programación son R y Python. R es mi lenguaje para datos estáticos, mientras que Python me permite integrarme en ecosistemas dinámicos y realizar tareas que requieren más ingeniería de software. Por ejemplo, extraer datos de una API o implementar un pipeline ETL.
